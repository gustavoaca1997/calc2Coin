# calc2Coin
## Calculator for Bolívares to various currencies, including Dollar and Dash
## Install
Clone the repository and move to the repository folder and run.
```bash
python init.py
```
## Use
Run
```bash
python calc2Coin.py
```
Wait for the data to be taken from the CoinmarketCap API or from the database. Then enter the current value of the Dollar in Bolívares and then the amount of Bolívares that you want to convert to the available currencies.

## Available currencies
1. Dollar
2. Dash
3. GBYTE / MBYTE
